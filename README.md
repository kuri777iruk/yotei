# YOTEI — 日程調整アプリ

カレンダーをタップするだけで候補日を選べる、シンプルな日程調整ツール。

## 機能

- 📆 カレンダーで複数の候補日を直感的に選択
- ⭐ ◎○△× の4段階優先度マーク
- ⏰ 9時から2時間おきの時間帯プリセット
- 🔒 回答期限オートロック（期限後は自動締切）
- 📧 日程確定時に参加者へメール通知
- 📅 Googleカレンダーとの連携ボタン

## 技術スタック

- HTML / CSS / JavaScript（フレームワーク不使用）
- 単一ファイル構成（`index.html`）

## デプロイ

[Vercel](https://vercel.com) でホスティング
